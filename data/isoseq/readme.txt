## Assembly of Iso-Seq data

Folders:

isoseq3_pipeline:
clustering of FLNC reads into clustered HQ transcripts

mapping_and_collapse:
mapping of clustered HQ transcripts into unique transcripts using the polished reference genome

mapping_and_collapse_old_genome:
mapping of clustered HQ transcripts into unique transcripts using the unpolished reference genome

comparison_genome_polishing:
comparison of the effect of genome polishing on collapse and completenes of transcript sequences

cpc2:
ORF prediction using CPC2

sqanti:
Reference-genome based correction of possible sequencing errors using SQANTI3
